Fitness Class Booking System

About the Project:
I developed a desktop-based Fitness Class Booking System using Java to streamline gym and studio operations. The application enables seamless management and scheduling for various fitness sessions (such as Yoga and Cardio) while handling trainer assignments and client bookings efficiently. 

I started the project from scratch, beginning with system design and UML modeling to structure the architecture cleanly. I heavily implemented Object-Oriented Programming (OOP) principles and designed an interactive Graphical User Interface (GUI) to deliver a user-friendly and responsive booking experience.

Technical Skills I Applied:
- Object-Oriented Programming (OOP): I utilized core OOP concepts to build robust and scalable classes for managing classes, trainers, and members.
- GUI Development: I built an intuitive graphical interface to ensure a smooth navigation and real-time booking process for the users.
- System Design & Logic: I engineered the backend logic to handle schedules efficiently and prevent any booking conflicts or data overlapping.
